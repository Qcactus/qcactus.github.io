Underscores in Python

Underscores in Python

Naming with Underscores in Python - Medium

Single Leading Underscore

_var Naming convention. Indicating name is meant for internal use.

Double Leading Underscore

__var Triggers name mangling when used in class context. Enforced by the Python interpreter.

Single Trailing Underscore

var_ Naming Convention. Avoid naming conflicts with Python keywords.

Double Trailing Underscore

__var__ Naming Convention. Indicates special methods defined by Python language.

Underscore

_ Naming Convention. Used as a name for temporary variables.
