2020-02-09

Overview of Neural Symbolic Reasoning

Why

neural networks: hard to train, not robust, not interpretable

Neural Programmer

  • table-comprehension dataset, arithmetic and logic questions
  • built-in operations -> compositional programs
  • at each step, select one segment in data source and one operation to apply to that segment
  • end-to-end backpropagation

Neural Programmer-Interpreters

  • single recurrent core shared by many programs, domain-specific encoders
  • learnable subprograms
  • curriculum learning
  • trained with fully-supervised execution traces

Neural Module Networks

  • VQA
  • trainable small modules, jointly trained
  • the structure of modular networks is determined by the stucture of the question
  • later work also learn the network structures by using reinforcement learning
  • the third paper, end-to-end training, predict network structures specialized for each question

Neural Symbolic Machines

  • seq2seq model to map the question to a program
  • reinforcement training

Neural-Symbolic VQA

  • disentangles vision and language understanding from reasoning, scene representation

Neuro-Symbolic Concept Learner

  • scene understanding and semantic parsing are trained jointly

  • can learn new concepts

Neural State Machine

  • predict a scene graph for an image
  • perform sequential reasoning over the graph

Summary

  • submodules (trainable or not) put together to be a compositional model
  • the structure can be trainable or not trainable
  • training methods: vision and language understanding and reasoning can be trained seperately or jointly; end-to-end training or every step gets supervised; reinforcement; curriculum learning
  • generalization: cross-domain, learn new concepts
  • differentiable
